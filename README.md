Machine Learning for Healthcare – Patient Segmentation

This project focuses on patient segmentation using clustering techniques to help healthcare organizations better understand patient groups and improve decision-making.

Overview
Using a real-world dataset of 2,000 patient records, this project applies unsupervised machine learning to group patients based on demographics, clinical conditions, insurance, and visit patterns.

Key Concepts
Clustering (Unsupervised Learning)
Distance-based segmentation
Handling mixed healthcare data
AI-assisted code generation & validation

Tech Stack
Python (Pandas, Scikit-learn)
Data Visualization
AI-assisted development

Outcomes
Segmented patients into meaningful clusters
Identified patterns in patient behavior and conditions
Demonstrated importance of algorithm selection for real-world data
